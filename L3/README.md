3. Скрипт backup.sh (Bash)
Возможности:
Принимает путь к папке как параметр

Поддерживает -h и --help для вывода справки

Проверяет существование исходной папки

Ищет изображения рекурсивно (jpg, jpeg, png, gif, heic)

Создает папку резервной копии с именем <папка>_backup_<дата>

Выводит процесс копирования с нумерацией

При отсутствии изображений удаляет пустую папку

Использованные конструкции:
Конструкция	Назначение
#!/bin/bash	Шебанг — указание интерпретатора
show_help()	Функция вывода справки
$1	Первый параметр скрипта
-z "$1"	Проверка, что параметр не пустой
dirname / basename	Получение родительской папки и имени папки
date +"%Y%m%d-%H%M%S"	Формирование timestamp
[ ! -d "$SRC" ]	Проверка существования папки
mkdir -p	Создание папки (с родителями)
$?	Код возврата последней команды
find -type f -iname	Рекурсивный поиск файлов (регистронезависимо)
-print0 и IFS= read -r -d ''	Безопасная обработка файлов с пробелами
while IFS= read -r -d '' file	Цикл чтения найденных файлов
cp	Копирование файла
rmdir	Удаление пустой папки
exit 0/1	Код выхода (успех/ошибка)
4. Скрипт backup.ps1 (PowerShell)
Возможности:
Принимает путь к папке как параметр

Поддерживает -h и --help для вывода справки

Проверяет существование исходной папки

Ищет изображения рекурсивно (jpg, jpeg, png, gif, heic)

Создает папку резервной копии с именем <папка>_backup_<дата>

Выводит процесс копирования с нумерацией

При отсутствии изображений удаляет пустую папку

Использованные конструкции:
Конструкция	Назначение
param([string]$Path)	Объявление параметра скрипта
function Show-Help	Функция вывода справки
Write-Host	Вывод цветного текста в консоль
if (-not $Path)	Проверка, что параметр передан
Test-Path -PathType Container	Проверка существования папки
Split-Path -Parent/-Leaf	Разделение пути на части
Get-Date -Format	Формирование timestamp
Join-Path	Объединение частей пути
New-Item -ItemType Directory	Создание папки
try/catch	Обработка ошибок
Get-ChildItem -Recurse -Filter	Рекурсивный поиск файлов
foreach	Цикл перебора файлов
Copy-Item	Копирование файла
Remove-Item	Удаление папки
exit 0/1	Код выхода
